PROJECT OVERVIEW:
This project focuses on analyzing retail sales performance across multiple retailers and beverage brands.
The primary objective was to assess revenue, units sold, and payment performance to identify high-performing retailers and brands, and to uncover insights that support better business decision-making.
The analysis was conducted using Microsoft Excel, leveraging Pivot Tables, KPIs, slicers, and interactive dashboard design principles.

❖Key Insights:
1- Costco generated the highest total revenue among retailers.
2- Coca-Cola recorded the highest units sold compared to other beverage brands.
3- Revenue distribution shows a clear performance gap between retailers, indicating opportunities for performance benchmarking.
4- The average payment cycle (31 days) highlights overall collection efficiency and potential cash flow impact.

❖Challenges & Improvements:
1- KPI Implementation Initially, the dashboard lacked executive summary indicators. After adding KPIs, the dashboard became more decision-focused and management-friendly.
2- Dashboard Design & Color Optimization: Improved color consistency and visual hierarchy to make insights clearer and more professional.

❖Lessons Learned:
1- The importance of KPIs in executive dashboards.
2- How visual hierarchy and layout impact insight clarity.
3- Structuring dashboards for decision-makers, not just for data display.
4- Transitioning from a functional dashboard to a business-oriented dashboard.
